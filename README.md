| Prefix (type) | 설명                |
| ------------- | ----------------- |
| `feat/`       | 새로운 기능 개발         |
| `fix/`        | 버그 수정             |
| `refactor/`   | 리팩토링 (기능 변경 없음)   |
| `style/`      | 스타일 (UI, CSS 등)   |
| `docs/`       | 문서 변경             |
| `test/`       | 테스트 코드 추가         |
| `chore/`      | 설정, 빌드 수정 등 기타 작업 |
